Using the install Script

curl --proto '=https' --tlsv1.2 -sSfL https://install.prasadt.com/gfilt | bash -s --

To upgrade existing binary usem

curl --proto '=https' --tlsv1.2 -sSfL https://install.prasadt.com/gfilt | bash -s --upgrade

To install exact version use --version <VERSION_TAG>.

curl --proto '=https' --tlsv1.2 -sSFL https://install.prasadt.com/gfilt | bash -s -- --version v0.2.1

By default we will only install for the user, who is running the script. If you wish to install for all users on the system, simpley run the script as root.

curl --proto '=https' --tlsv1.2 -sSFL https://install.prasadt.com/gfilt | sudo bash -s --

To see more available options like skipping checksum verification and skip installing manpages etc, run the script with -h or --help flag.

curl --proto '=https' --tlsv1.2 -sSFL https://install.prasadt.com/gfilt | sudo -s -- help
